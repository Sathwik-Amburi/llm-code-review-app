# Code Review APP for LLM Generated Code Snippets

## Setup

1. Clone the repository
2. Install the dependencies
3. Run the app

```bash
git clone https://github.com/Sathwik-Amburi/llm-code-review-app.git
cd code-review-app
yarn install
yarn run dev # for development
yarn run build # for production
yarn run start # for production
```
